## 🚀 **Comment utiliser maintenant :**

### 1. **Créez vos tags :**
```
Menu "Tag" → Créer :
- Nom: Protagoniste
- Catégorie: Personnages
- Couleur: #4F46E5

- Nom: Arme
- Catégorie: Objets  
- Couleur: #DC2626

- Nom: Royaume
- Catégorie: Régions
- Couleur: #059669
```

### 2. **Ajoutez les tags à vos fiches :**
- **Travis** → Tags → "Protagoniste"
- **Épée magique** → Tags → "Arme"
- **Royaume de Flamme** → Tags → "Royaume"

### 3. **Filtrez facilement :**
- Tous les protagonistes
- Toutes les armes
- Tous les royaumes
